# ☕ Alina's Café - Full Stack Mobile App

This is a full-stack café ordering mobile application built using React Native and Node.js.

The app allows customers to:
- View menu items
- Add items to cart
- Place orders

The manager can:
- View incoming orders

---

 Technologies Used

Frontend
- React Native
- Expo
- Expo Router
- Axios

 Backend
- Node.js
- Express.js

 How to Run

Backend
1. cd backend
2. npm install
3. node server.js

Frontend
1. cd frontend
2. npm install
3. npm start
4. Scan QR code using Expo Go
